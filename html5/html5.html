About HTML5 Starter Kit

Author: Bilal Çınarlı | January 8th, 2010 | Documentation, Tutorials | 123 Comments

HTML5 Starter kit is a basic starting point for your psd to html process. It contains most of common elements that are present in every web page. You can use this as a base for your website or wordpress theme.

Properties

  1. It has all starting needs with a global reset (not a really reset perhaps), layout definitions and text definitions with most used heading (h1 to h3), ordered and unordered lists, paragraph and link styles
  2. HTML layout consist of a header section, navigation, main column, sidebar and footer
  3. You can changed the easily change or move the part such as seach box, navigation by copying and pasting to them to another section like from header to sidebar.
  4. It has a favicon definition by default. Just prepare a 16x16px or 32x32px sized png image named as favicon.png and put it into images directory
  5. I did not provide any images, since this document should be the layout template for your work
  6. By default both jQuery and MooTools javascript frameworks are added at the very end of the page from google jsapi. Just delete the definition line which framework you're not going to be use. Also if you want to use local files for these frameworks, you should remove google jsapi line and add <script type="text/javascript" src="scripts/your_local_js_framework_file.js"></script>
  7. If you want you can use HTML4 version of this kit which is also present in the rar file named as index.html.
